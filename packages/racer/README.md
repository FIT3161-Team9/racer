# Racer

This package is the C++ source code for the Racer simulation-based game.

## Development

### Requirements

- CMake (Version 3.10 or greater)
- A C++20 compliant compiler (Not all C++20 features are required)
- An install of SFML

### Building & Runing

From the package library:

- `mkdir build`
- `cd build`
- `cmake ..`
- `cmake --build .`
- `./RACER`
